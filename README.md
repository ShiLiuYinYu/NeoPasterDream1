## 📖 模组简介 / Overview

该模组是帕斯特之梦在 1.21.1 NeoForge 加载器上的移植。

***

## ⚙️ 开发状态 / Development Status

> 🚧 **当前阶段：核心框架搭建 + 染梦维度基础地形完成**

| 模块 | 状态 | 说明 |
|:----|:----:|:----|
| **项目基础** | ✅ 完成 | NeoForge 1.21.1 MDK 搭建，物品/方块/实体注册体系 |
| **染梦维度** | ✅ 完成 | 4种生物群系（草原/炎热/寒冷/海洋），洞穴峡谷生成正常 |
| **维度传送** | ✅ 完成 | 传送水晶 （暂用） |
| **BiomeModifier 框架** | ✅ 完成 | 代码级特征注入系统（`PDDyedreamBiomeModifier`），便于后续扩展 |
| **模型/纹理修复** | ✅ 完成 | 适配 1.21.1 模型格式，修复墙/栅栏/纹理 WARN |
| **矿物生成** | ⏳ 待配置 | 3种矿石配置特征已就绪，待通过 BiomeModifier 激活 |
| **植被/树木** | ⏳ 待添加 | 染梦草、睡莲、莲花、海草、染梦树等装饰特征待注入 |
| **染梦能量系统** | 🔄 开发中 | San值/能量机制实现 |
| **更多维度内容** | 📅 计划中 | 灯影世界、风之旅维度等后续移植 |

### 🔧 近期修复记录
- **世界创建卡死** — 修复 biome JSON 重复 `features` 键 + 适配 1.21.1 `configured_feature` 格式（`predicate_type` + `tag_match`）
- **模型 WARN** — 墙模型 parent 改为 `template_wall_*`，栅栏 blockstate 改为 `multipart` 格式
- **纹理占位** — 生成 `大地之翼` / `尖牙之翼` 16x16 像素纹理
